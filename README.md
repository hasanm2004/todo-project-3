To-Do List Application:

A simple and interactive To-Do List Application built using HTML, CSS, and JavaScript. This project allows users to add and delete tasks dynamically, demonstrating the use of DOM manipulation and event handling in JavaScript.

Features:
➕ Add new tasks dynamically
🗑️ Delete tasks instantly
⚡ Real-time task management
🎨 Clean and user-friendly interface
📱 Responsive design
🔄 Dynamic DOM manipulation
🖱️ Interactive button functionality

Technologies Used:

Project Objective:

The goal of this project was to understand how JavaScript can make web pages interactive by allowing users to dynamically add and remove content without refreshing the page.
